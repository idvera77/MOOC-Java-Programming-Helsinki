# Java Programming Course - University of Helsinki

Este repositorio contiene mis soluciones a los ejercicios del prestigioso curso **Java Programming I & II** de la Universidad de Helsinki (MOOC.fi).

## Tecnologías y Conceptos Dominados

A través de este curso, he desarrollado una base sólida en el desarrollo de software con Java, enfocándome en:

* **Programación Orientada a Objetos (OOP):** Encapsulamiento, herencia, polimorfismo e interfaces.
* **Estructuras de Datos:** Manejo avanzado de `ArrayList`, `HashMap` y procesamiento de colecciones.
* **Lógica de Negocio:** Separación clara entre la lógica del programa y la interfaz de usuario (UI).
* **Algoritmos:** Implementación de sistemas de búsqueda, filtrado y gestión de datos.
* **Clean Code:** Escritura de código legible, organizado y siguiendo las convenciones de Java.

## Estructura del Repositorio

* **Part 01-07:** Fundamentos de Java, objetos, listas y lógica de aplicaciones (Completado).
* **Part 08-14:** Conceptos avanzados, HashMaps, Streams y archivos (En progreso...).
